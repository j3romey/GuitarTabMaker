READ ME 

open terminal
locate .jar file
type: java -jar GuitarTabMaker.jar

1. Enter the name of Song
2. a) 	Enter the fret on the corresponding string
		indicated at the left side
	b) 	Repeat for all string EADGBe
	c)	After you are given 3 choices
		n: move to the next line with new inputs
		c: copy the current inputs to the next one
		d: finish the program and print to to test.docx
		   you should see the file in the folder with the jar in it.
		
Valid inputs: 0-9 (need to add double digit numbers), /, \ , h, p
